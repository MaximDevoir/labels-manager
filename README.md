# Probot Labels

Probot-Labels makes managing your project's GitHub labels easy and hassle-free.

## Resources

[GitHub Labels API V3](https://developer.github.com/v3/issues/labels/#update-a-label)

## Features

- Updates labels on changes to certain files.
